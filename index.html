PIPEBOT

(In progress) Pipe-inspection robot has great potential in diverse industries including oil and gas for tasks such as detection of defects in pipelines. The main purpose of our project is to design a robot to (1) recognize three types of pipe topologies: U, T, and Y shaped, and (2) successfully move through those topologies. Challenges involved in achieving such goals include designing and implementing an appropriate sensor system, feedback control system, as well as various algorithms. See the OEDK website for more information

ROS, Beaglebone, LIDAR, Servomotor, Solidworks, Python, etc.

A Safe Safe

This was a hackathon project developed during HackRice 2016 in 36 hrs (Repository). It allows the user to lock and unlock a safe from his/her phone. It also allows the user to check if the safe is locked or unlocked as well as change the master password that secures the app. It also provides visual feedback (LED), audio feedback (buzzer), and physical feedback (servo moving a latch up and down to lock and unlock)

Raspberry Pi running TornadoWeb, Arduino with the ability to connect to a computer with Arduino IDE, Computer running PhoneGap software, Smartphone running PhoneGap Developer App

Rice Apps - Petitions Web App

I wrote an open-source web application (still in development) that allows: (1) organizations to create elections with a time-frame and positions, (2) Rice students to create petitions to be on the ballot for those elections, (3) other Rice students to sign those petitions, and (4) email notifications to the petitioner and the organization administrators when the required number of signatures have been met.
